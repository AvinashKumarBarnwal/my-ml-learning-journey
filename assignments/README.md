# Machine Learning Assignments

This directory contains topic-wise Machine Learning assignments completed as part of my AI & ML learning journey and academic coursework.

Each assignment is organized as an independent mini-project containing:
- Problem Statement
- Dataset
- Jupyter Notebook Implementations
- Python Scripts
- Visualizations
- Notes and Observations
- Model Evaluation and Conclusions

The objective of maintaining this structure is to document the complete end-to-end Machine Learning workflow in a clean, organized, and reproducible manner.
