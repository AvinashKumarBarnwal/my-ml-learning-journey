# Assignment 01 — Store Sales Prediction using Linear Regression

## Overview

This assignment focuses on understanding the complete workflow of a Machine Learning regression problem using Linear Regression techniques.

The objective is to explore not only model building, but also the practical steps involved in preparing real-world data for machine learning.

- Creating meaningful feature representations

## Linear Regression Modeling
- Understanding regression workflow

- Prediction generation
- Learning model behavior

- Error analysis
- Performance evaluation
- Understanding regression metrics
