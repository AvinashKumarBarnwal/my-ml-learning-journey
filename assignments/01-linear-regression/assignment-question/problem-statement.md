# Assignment 01 — Store Sales Prediction using Linear Regression

## Problem Statement

A supermarket chain wants to predict the sales revenue of its stores based on multiple business factors such as:

- Advertising Budget
- Store Size
- Number of Employees
- Product Categories
- Customer Footfall
- Promotional Offers

A retail sales dataset is provided for analysis.

The goal of this assignment is to perform:
- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Feature Engineering
- Linear Regression Modeling
- Model Evaluation

to predict store sales effectively.

---

# Assignment Tasks

## 1. Exploratory Data Analysis (EDA)

Perform Exploratory Data Analysis on the dataset.

### Objectives
- Identify missing values
- Generate descriptive statistics
- Understand feature distributions
- Analyze relationships between variables

### Required Visualizations
- Histogram
- Correlation Heatmap
- Scatter Plot / Box Plot

### Expected Outcome
Write observations and inferences based on the generated visualizations.

---

## 2. Data Preprocessing and Feature Engineering

Perform necessary preprocessing steps such as:

- Handling missing values
- Encoding categorical variables
- Feature scaling (if required)

### Expected Outcome
Explain the preprocessing techniques and feature engineering approaches used.

---

## 3. Machine Learning Model Building

Build a Linear Regression model for store sales prediction.

### Points to Explain
- Features selected
- Target variable
- Regularization method used (if any)
- Reason for selecting Linear Regression

---

## 4. Model Evaluation

Evaluate the model using suitable regression metrics such as:

- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)
- R² Score

### Expected Outcome
Interpret the evaluation metrics and assess overall model quality.

---

## 5. Observation and Conclusion

Write observations from the complete workflow and conclude the effectiveness of the Linear Regression model for predicting store sales.
